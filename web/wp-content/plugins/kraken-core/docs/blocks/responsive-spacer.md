# Responsive Spacer Block

_Last Updated: October 2025_

## Overview

The Responsive Spacer block provides flexible spacing control across different screen sizes. It's perfect for creating responsive layouts with appropriate spacing for desktop, tablet, and mobile devices.

## Block Attributes

| Attribute       | Type   | Default | Description              |
| --------------- | ------ | ------- | ------------------------ |
| `heightDesktop` | string | "4rem"  | Spacer height on desktop |
| `heightTablet`  | string | "4rem"  | Spacer height on tablet  |
| `heightMobile`  | string | "3rem"  | Spacer height on mobile  |

## Best Practices

### Spacing Guidelines

- **Desktop**: Use larger spacing (4-8rem) for better visual hierarchy
- **Tablet**: Use medium spacing (3-6rem) for balanced layout
- **Mobile**: Use smaller spacing (2-4rem) to maximize content visibility
