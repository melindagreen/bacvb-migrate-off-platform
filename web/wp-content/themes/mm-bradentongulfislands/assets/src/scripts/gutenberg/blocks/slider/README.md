# Slider Block

An elaborate slider gutenberg block using SwiperJS. It allows importing of posts, custom items or innerblocks.
## Installation

You just need to add swiper to your package.json by one of the following commands;

```bash
yarn add swiper
npm install swiper
```

## Usage

```node
// import Swiper bundle with all modules installed
import Swiper from 'swiper/bundle';

import 'swiper/css/bundle';

// init Swiper:
const swiper = new Swiper(...);
```

```scss
// import styles bundle in styles/style.scss & styles/index.scss
@import "swiper/css/bundle";
```

Learn more at [SwiperJs](https://swiperjs.com/get-started)
